Proyecto Destacado

AgroSOS - Plataforma de Gestión Agrícola

Proyecto final de Ciclo Superior para el Ministerio de Agricultura

AgroSOS es una aplicación web integral diseñada para la monitorización y gestión remota de parcelas agrícolas. La plataforma permite a los agricultores optimizar el uso de recursos, automatizar tareas y tomar decisiones informadas basadas en datos en tiempo real recogidos por sensores IoT.

Funcionalidades Clave:

  • Visualización de datos de sensores (temperatura, humedad, etc.) en gráficos dinámicos.
  • Gestión completa de parcelas, cultivos, sensores y actuadores. Sistema de automatización basado en reglas personalizables para el control del riego.
  • Autenticación de usuarios con roles (administrador y agricultor) mediante JWT.
  • Interfaz multi-idioma (español e inglés) para una mayor accesibilidad.
